HowToInstallUbuntuFromDifferentOSWithoutCD¶

Download the following files to /boot on a partition other than the one you will install to. Ideally use a different hard drive.
  • http://archive.ubuntu.com/ubuntu/dists/hardy/main/installer-i386/current/images/hd-media/initrd.gz
  • http://archive.ubuntu.com/ubuntu/dists/hardy/main/installer-i386/current/images/hd-media/vmlinuz

Place the server install CD ISO in the /boot directory.

Update GRUB to boot from that partition.

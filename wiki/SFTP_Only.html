SFTP Only¶

Server Setup¶

Create a sftponly group.

sudo addgroup sftponly

Edit /etc/ssh/sshd_config

Alter the Subsystem sftp to internal-sftp

Subsystem sftp internal-sftp

Append

Match group sftponly
  ChrootDirectory %h
  X11Forwarding no
  AllowTcpForwarding no
  ForceCommand internal-sftp

Per User¶

  • Create user
  • Add user to sftponly group
  • Set user's home directory to be owned by root.

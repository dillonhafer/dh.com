HowToTunnelXWithSSH¶

As of Ubuntu 8.04 all you have to do is use the -X parameter with SSH. Then just run a X program.

To run Firefox remotely use the following command or it will still open a local instance:

firefox -no-remote &

To go from machine A -> B -> C set up port forwarding:

ssh -L 2222:hostc:22 hostb

Then in another terminal
ssh -p 2222 -X localhost

On Windows install XMing.

More info at:
  • http://www.cag.lcs.mit.edu/~wentzlaf/faq/ssh_X.html - basics
  • http://linux.ucla.edu/pipermail/linux/2002-October/007688.html - A -> B -> C
  • http://gears.aset.psu.edu/hpc/guides/xming/ - from Windows

There is also an attached ruby/applescript to do this for you.

Linux and Samba commands¶

Samba¶

adduser just to samba: sudo adduser --disabled-password username
Change Password: sudo smbpasswd -a username
Delete user: sudo smbpasswd -x username
List users: sudo pdbedit -w -L

Ubuntu¶

Add Use: sudo adduser username
Delete User: sudo userdel -r username

find¶

Find all files ending in .txt case insensitively in current directory:

find . -iname '*.txt'

Find all files ending in .txt case insensitively in current directory and do something with the file (cat in this case):

find . -iname '*.txt' -exec cat {} \;

Find broken symlinks in current directory.

find -L . -type l

Root¶

This works for any flavor of *nix

1. Enter single-user mode. (OSX CMD+s at startup, ubuntu is usually SHIFT)

The following steps pertain to OSX specifically because default doesn't mount / as writable.
2. fsck -y
3. mount -uw /

Once you have write access to the drive (on any platform) run the following:
4. visudo

Now just give your user the proper permissions.

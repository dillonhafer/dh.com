Ubuntu listening on multiple IP's¶

See example below. Note in particular that the aliases do not have broadcast addresses. This can cause them not to start at boot time. See bottom of https://bugs.launchpad.net/debian/+source/ifupdown/+bug/114457

# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
auto eth0
iface eth0 inet static
    address 10.250.2.249
    netmask 255.255.0.0
    network 10.250.0.0
    broadcast 10.250.255.255
    gateway 10.250.2.2
    # dns-* options are implemented by the resolvconf package, if installed
    dns-nameservers 10.250.2.254
    dns-search hylesanderson.edu

# students
auto eth0:0
iface eth0:0 inet static
        address 10.250.2.81
        netmask 255.255.0.0
        network 10.250.0.0

# online
auto eth0:1
iface eth0:1 inet static
        address 10.250.2.247
        netmask 255.255.0.0
        network 10.250.0.0

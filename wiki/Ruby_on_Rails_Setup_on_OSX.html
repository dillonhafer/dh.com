Ruby on Rails Setup on Mac OSX¶

Prerequisites¶

1. Mac OS X >= 10.7
2. The latest Xcode Tools from the Mac App Store
3. GIT
4. RVM

GIT¶

Install the GIT installer from

http://git-scm.com

RVM¶

Download with the following command.

user$ bash < <(curl -s https://raw.github.com/wayneeseguin/rvm/master/binscripts/rvm-installer)

Install by adding this to .bash_profile or equivalent.

user$ echo '[[ -s "$HOME/.rvm/scripts/rvm" ]] && . "$HOME/.rvm/scripts/rvm" # Load RVM function' >> ~/.bash_profile

Ruby¶

OSX comes pre-installed with ruby 1.8.7, but is not suitable for Rails 3 applications.

Install ruby with RVM

rvm get head
rvm reload
rvm install 1.9.x

Set Ruby 1.9.x as default ruby version

rvm use 1.9.x --default

Install Ruby 1.9.x from source if you cannot use RVM

Ruby 1.9.2 (used version number as example)¶

Download Ruby source code 1.9.2 p290 (or the latest version of the 1.9.2 branch)

curl -0 ftp://ftp.ruby-lang.org//pub/ruby/1.9/ruby-1.9.2-p290.tar.bz2
curl -0 http://files.rubyforge.vm.bytemark.co.uk/rubygems/rubygems-1.3.5.tgz

By default this will install in /usr/local/bin, /usr/local/lib

tar -xzvf ruby-1.9.2-p290.tar.bz2 
cd ruby-1.9.2-p290/
./configure
make
sudo make install

Update Rubygems

sudo gem update --system

Rails¶

Install rails

sudo gem install --no-rdoc --no-ri rails

Install native PostgreSQL-Ruby bindings (if using PostgreSQL)

sudo gem install --no-rdoc --no-ri pg

Use the one-click installer for installing PostgreSQL

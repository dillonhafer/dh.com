Base Ubuntu Server Setup¶

Run standard server install. In the disk partitioning step accept the default of LVM. When choosing packages to install only choose OpenSSH.

Install and initialize etckeeper

sudo aptitude install git-core etckeeper

Edit /etc/etckeeper/etckeeper.conf and change VCS from bzr to git

sudo etckeeper init
sudo etckeeper commit "initial import"

Set up SSH key authentication. Simplest way is to copy the authorized_keys file from a working server to ~/.ssh/authorized_keys.

Disallow password logins via SSH. Edit /etc/ssh/sshd_config. Set PasswordAuthentication to no

Update package list and install updates

sudo aptitude update
sudo aptitude full-upgrade

Install standard extra packages

sudo aptitude install htop iotop mc diakonos ack-grep tmux mtr dstat acpid

Make sure time is kept synced automatically

sudo aptitude install ntp

Install postfix and setup according to HowToInstallPostfixAsSecureRelay

How To Performance Test Network With Linux¶

Install iperf on two machines. Make sure the server's firewall allows port 5001.

Server:

jackc@db-replicant:~$ iperf -s
------------------------------------------------------------
Server listening on TCP port 5001
TCP window size: 85.3 KByte (default)
------------------------------------------------------------
[  4] local 10.250.2.72 port 5001 connected with 10.250.2.188 port 34124
[  4]  0.0-10.3 sec    115 MBytes  94.1 Mbits/sec

Client:

jackc@jackc-pc:~$ iperf -c db-replicant
------------------------------------------------------------
Client connecting to db-replicant, TCP port 5001
TCP window size: 16.0 KByte (default)
------------------------------------------------------------
[  3] local 10.250.2.188 port 34124 connected with 10.250.2.72 port 5001
[  3]  0.0-10.1 sec    115 MBytes  96.0 Mbits/sec
